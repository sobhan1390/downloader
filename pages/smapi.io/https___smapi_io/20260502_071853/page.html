What's new

See what's new in SMAPI 4.5 for the main changes.

SMAPI 4.5 lets you configure SMAPI itself in-game, improves performance, adds automated and attested builds, retires the 'SMAPI for developers' edition, and adds malicious file detection.

Patch releases:

  • 4.5.1 fixes an error installing SMAPI 4.5.0 on Linux/macOS, and updates Portuguese translations.
  • 4.5.2 mainly improves performance, fixes the Linux/macOS installer not saving the color scheme correctly in 4.5.0+, and updates translations for recent changes.

Requires Stardew Valley 1.6.14 or later on Linux/macOS/Windows or Steam Deck.

(⚠️ For macOS players: don't forget to follow the full install guide for macOS to avoid security warnings. Players on Linux or Windows can ignore this.)

See the release notes and mod compatibility list for more info.
