Polish Translation for Stardew Valley

Description

[ENG]
This is a Polish translation for base game.

Translation is compatibile with the newest version of the game(1.6.15).
On our discord server you can report any mistakes found and download translation that doesn't need smapi.

DNKSH Team

-----> Instalation<-----
Remove older version before updating
(Tutorials in the form of short videos can be found on our discord server)
1.Download smapi.
2. Download Content Patcher
3. Download newest version of translation.
(Optional: Download Generic mod config menu to change font)
4. Unpack smapi(wersja) installer onto desktop and run file install on Windows.bat (On windows) or Install on Linux.sh (On Linux) or install on macOS.command (On Mac)
5. When black window appears it'll say: "Where do you want to add or remove smapi?" If next to "1" will be path in where you have installed your game press "1" and click enter, if not, press "2" and after that you need to paste the path where you have your game installed. (Where Stardew Valley.exe is)
4. Put folder [CP]Spolszczenie and content patcher to folder mods.
5. Start your game remembering to start it with StardewModdingApi.exe or by putting command to your launch options (on steam).
6. Choose Polish language by clicking small arrow in the language menu.

-----> Instalacja Spolszczenia <-----
Przed instalacją spolszczenia usuń starą wersje
(Poradniki w formie krótkich filmików są na naszym discordzie)
1. Pobierz smapi
2. Pobierz Content Patcher
3. Pobierz najnowszą wersje spolszczenia.
(Opcjonalne: Pobierz Generic mod config menu żeby zmienić czcionkę)
4. Wypakuj folder smapi(wersja) installer na pulpit i włącz plik install on Windows.bat (na systemie windows) lub Install on Linux.sh (Na systemie Linux) lub install on macOS.command (Na systemie Mac)
5. Wyskoczy ci czarne okienko, w którym będzie pisało: "Where do you want to add or remove smapi?" Jeśli przy "1" pojawia się ścieżka, gdzie masz zainstalowaną grę, to wpisujesz "1" jeśli nie, to wpisujesz "2" i później wklejasz ścieżke swojej gry. (Gdzie masz Stardew Valley.exe)
4. Włóż folder spolszczenie i content patcher do folderu mods.
5. Uruchom gre pamiętając aby ją uruchamiać poprzez StardewModdingApi.exe, albo wkleić komendę w opcji uruchamiania (Steam)
6. Wybierz język polski klikając małą strzałkę w prawo w menu języków
